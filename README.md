# Este projeto sobre Autômatos tem as seguintes implementações:

##1. Representação:
	> Estados
	> Estado inicial
	> Estados marcados
	> Eventos
	> Funções de transições

##2. Ações:
	> Adicionar estados
	> Adicionar eventos
	> Remover estados
	> Remover estados

##3. Operações com 1 autômato:
	> Acessível
	> Coacessível
	> Trim
	> Complemento

##4. Operações com 2 autômato:
	> Produto
	> Composição paralela

##5. Ações do programa:
	> Criar novo autômato
	> Caso de teste
	> Importar autômato
	> Exportat autômato
	> Sobre
